# dalmation
A GTK4 File Manager for Niri. This is a file manager for Niri written in Rust. The goal is to help me learn Rust and Linux GUI programming while building a tool which meets my exact needs from a file manager - image thumbnails, mouse selection and simple create/delete/copy file operations.

I want the UI to match the aethetic of Niri with Noctalia, including system theming and transparency.

A file manager is a simple concept to understand and will hopefully help me learn Rust (and GTK programming).
